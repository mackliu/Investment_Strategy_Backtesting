A.1 Block Bootstrap 信賴區間A.1 Block Bootstrap Confidence Intervals

Block Bootstrap 是用來建立統計信賴區間的穩健方法。 由於滾動報酬具有高度自相關，傳統的「均值 +/- 1.96 x 標準誤」方法會產生過窄的信賴區間。 Block Bootstrap 以 12 個月為區塊重新抽樣（保留短期時間結構），重複 10,000 次， 從而得到更可靠的 95% 信賴區間。

如何閱讀：「基準在 CI 內」欄位為 Yes 時，表示我們無法在 95% 水準上 區分觀察到的均值與基準值——換言之，觀察到的報酬在統計上與基準「一致」。 反之，若為 No，則有 95% 的信心認為真正的長期報酬與基準不同。

Block Bootstrap is a robust method for confidence interval construction. Rolling returns' high autocorrelation makes traditional "mean +/- 1.96 x SE" intervals too narrow. Block Bootstrap resamples in 12-month blocks (preserving short-term structure), repeated 10,000 times, producing more reliable 95% confidence intervals.

How to read: When "Benchmark in CI" is Yes, we cannot distinguish the observed mean from the benchmark at 95% level — the observed return is statistically "consistent" with the benchmark. If No, we have 95% confidence that the true long-term return differs from the benchmark.

A.2 歷年年度報酬A.2 Annual Returns by Year

下圖展示 1872-2022 年的完整年度報酬序列（共 151 年）。 正報酬以綠色表示，負報酬以紅色表示，10% 基準線以虛線標示。

觀察重點：(1) 報酬的波動性極大——單年報酬從 -40% 到 +50% 不等； (2) 負報酬年份的頻率約為 28%（約每 3.6 年出現一次）； (3) 大部分年份的報酬低於 10%——「10%」看似是中間值，但實際上是被少數極端正報酬年份拉高的均值； (4) 1930s 和 2000s 的密集紅柱清楚展示了「失落年代」的衝擊。

The chart below shows complete annual returns from 1872-2022 (151 years). Green for positive, red for negative returns. 10% benchmark as dashed line.

Key observations: (1) Extreme volatility — single-year returns range from -40% to +50%; (2) Negative return years occur ~28% of the time (about once per 3.6 years); (3) Most years fall below 10% — "10%" seems like a midpoint but is actually a mean pulled up by extreme positive years; (4) Dense red bars in 1930s and 2000s clearly show "lost decade" impacts.

A.3 月報酬分佈A.3 Monthly Return Distribution

月報酬的分布特性是理解長期投資風險的基礎。 S&P 500 月報酬的統計特性顯示：(1) 偏態 = 0.00（正偏——右尾較長， 代表少數月份有極高報酬）；(2) 峰態 = 0.0（遠高於常態分布的 3—— 「厚尾」意味著極端事件出現的頻率遠高於常態假設）。

厚尾分布在投資實務上的意義：用常態分布模型（如 VaR）來估計風險會嚴重低估 極端損失的機率。2008 年 10 月的 -17% 月跌幅在常態假設下幾乎不可能發生， 但在厚尾分布中是合理的尾部事件。

Monthly return distribution characteristics are foundational for understanding long-term risk. S&P 500 monthly returns show: (1) skewness = 0.00 (positive — right tail longer, meaning a few months with extremely high returns); (2) kurtosis = 0.0 (far above normal distribution's 3 — "fat tails" mean extreme events occur far more often than normal).

Fat tails in practice: risk models assuming normality (e.g., VaR) severely underestimate extreme loss probabilities. October 2008's -17% monthly drop would be near-impossible under normality, but is a reasonable tail event under the actual fat-tailed distribution.

This PNG chart shows long-term cumulative performance of international indices as a static supplement to the interactive chart. All indices normalized to 1988 = 100 for visual comparison.

Japan's lesson: The Nikkei 225's "lost 35 years" is clearly visible — peaking in late 1989 then declining, only gradually recovering in the 2020s. This is the strongest counterexample to "stocks always rise long-term." Note: Nikkei and FTSE are price indices (excluding dividends); adding estimated dividend reinvestment would raise Nikkei's actual cumulative return above what the chart shows, but still far below the US.

A.5 名目 vs 實質累積報酬A.5 Nominal vs Real Cumulative Returns

此圖以對數刻度呈現 1871 年至今的名目和實質累積報酬走勢。 對數刻度讓讀者能更直觀地比較不同時期的增長率—— 在對數圖中，斜率代表增長率，水平線段代表零增長。

觀察重點：(1) 兩條線的「距離」在 1970s 後明顯擴大—— 這是因為 1970s 的高通膨（年均 7%+）造成名目和實質報酬之間的大幅偏離； (2) 實質報酬線的斜率相對穩定，反映了美國經濟長期的「真實增長」； (3) 名目報酬線的陡峭斜率很大程度上是通膨的「錯覺」—— 這正是為什麼退休規劃必須使用實質報酬而非名目報酬。

This log-scale chart shows nominal and real cumulative returns from 1871 to present. Log scale enables intuitive growth rate comparison — slope represents growth rate, horizontal segments represent zero growth.

Key observations: (1) The "gap" between lines widened notably after the 1970s — high 1970s inflation (7%+ annual) caused major divergence between nominal and real; (2) The real return line maintains a relatively stable slope, reflecting the US economy's "true growth"; (3) The nominal line's steeper slope is largely inflation's "illusion" — this is precisely why retirement planning must use real, not nominal, returns.
