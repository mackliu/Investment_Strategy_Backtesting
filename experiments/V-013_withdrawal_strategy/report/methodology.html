2.3 提領策略定義2.3 Withdrawal Strategy Definitions

我們測試三種主流提領策略，每種代表不同的退休金管理哲學：

策略一：固定金額提領（傳統 4% 法則）

退休時計算初始年提領金額 = 投資組合 x 提領率（例如 1000 萬 x 4% = 40 萬/年），之後每年依 CPI 調整金額（確保購買力不變）。這是 Trinity Study 測試的原始方法。

優點：生活費用可預測，易於規劃。缺點：不考慮市場表現，熊市時持續提領相同金額加速資產消耗。

策略二：固定比例提領

每年提領當時投資組合市值的固定比例（例如 4%）。市值上漲時多領、下跌時少領。由於永遠只領現有資產的 X%，數學上永遠不會歸零（但金額可能非常微小）。

優點：永不破產（數學保證）。缺點：退休後期若市場不佳，實際提領金額可能低到無法維持基本生活。

策略三：動態調整 Guardrails

基準提領金額同固定金額法，但設定上下限：當投資組合表現良好時，最多可提領基準的 120%（上限，ceiling）；當表現不佳時，至少需降低至基準的 80%（下限，floor）。這個方法參考 Guyton & Klinger (2006) 的決策規則思路。

優點：市場好時享受更高生活品質，差時適度犧牲以保護資產壽命。缺點：生活費用有波動，需要更多財務紀律。

We test three mainstream withdrawal strategies, each representing a different retirement management philosophy:

Strategy 1: Fixed Amount (Traditional 4% Rule)

Calculate initial annual withdrawal = portfolio x rate (e.g., NTD 10000K x 4% = NTD 400K/year), then adjust annually by CPI (maintaining purchasing power). This is the original Trinity Study method.

Pros: Predictable living expenses, easy to plan. Cons: Ignores market performance; continued fixed withdrawals during bear markets accelerate depletion.

Strategy 2: Fixed Ratio

Withdraw a fixed percentage of current portfolio value each year (e.g., 4%). When markets rise, you withdraw more; when they fall, less. Since you always take X% of what remains, the balance mathematically never reaches zero (but amounts can become negligibly small).

Pros: Never goes bankrupt (mathematical guarantee). Cons: Late in retirement, actual withdrawals may be too low to sustain basic living.

Strategy 3: Dynamic Guardrails

Base withdrawal follows the fixed-amount method, but with ceiling and floor: when the portfolio performs well, withdraw up to 120% of the base (ceiling); when it underperforms, reduce to at least 80% (floor). This approach follows Guyton & Klinger (2006) decision rules.

Pros: Higher lifestyle when markets cooperate, modest sacrifice to protect portfolio longevity. Cons: Living expenses fluctuate; requires greater financial discipline.

2.4 模擬方法2.4 Simulation Methods

歷史路徑分析（Historical Path Analysis）

以美國 Shiller 數據為基礎，以每個月份為假想的退休起點，模擬 30 年（360 個月）的投資組合餘額變化。由於每年以 1 年為間隔移動窗口，共可產生 68 條不重疊的 30 年路徑。每條路徑代表「如果某位投資人在該年退休，他接下來 30 年會發生什麼事」。

為什麼需要這個方法？歷史路徑保留了報酬的時間序列結構（包括連續的牛市或熊市），直接告訴我們過去有多少比例的退休者會成功或失敗，是最直觀的分析方式。

Block Bootstrap 模擬

歷史路徑分析的最大限制是樣本量不足——即使有 150 年的美國數據，也只有 68 條 30 年路徑，而台灣數據更是完全無法產生 30 年路徑。Bootstrap 解決了這個問題。

Block Bootstrap 從歷史月報酬中以 12 個月為區塊（保留年內季節性與短期動量）隨機重新抽樣，拼接成 30 年的模擬路徑。重複 10,000 次，產生足夠大的樣本進行統計推論。

為什麼用 Block Bootstrap 而非簡單 Bootstrap？金融報酬存在短期自相關（動量效應），逐月隨機打亂會破壞這個時間結構。以 12 個月為區塊保留了年內的相關性，同時打亂年份間的順序以產生新的情境組合。

Historical Path Analysis

Using US Shiller data, we simulate portfolio balance evolution over 30 years (360 months) for each potential retirement start date. With annual window intervals, this generates 68 non-overlapping 30-year paths. Each path represents "what would happen if someone retired in that year."

Why this method? Historical paths preserve the time-series structure of returns (sustained bull/bear markets), directly showing what fraction of past retirees would have succeeded — the most intuitive analysis.

Block Bootstrap Simulation

The main limitation of historical paths is sample size — even 150 years of US data yields only 68 30-year paths, and Taiwan data cannot produce any. Bootstrap addresses this.

Block Bootstrap resamples from historical monthly returns in 12-month blocks (preserving intra-year seasonality and short-term momentum), assembling 30-year simulated paths. Repeating 10,000 times provides sufficient sample size for statistical inference.

Why Block Bootstrap over simple Bootstrap? Financial returns exhibit short-term autocorrelation (momentum). Shuffling individual months would destroy this structure. 12-month blocks preserve intra-year correlations while reshuffling inter-year sequences.

2.6 統計檢定方法2.6 Statistical Testing Methods

為回答四個研究問題，我們採用以下統計檢定方法：

  • Q1（台灣 4% 是否達 95%）：二項式檢定（Binomial Test）。將每次 Bootstrap 模擬的存活結果視為伯努利試驗（存活=1，破產=0），檢定成功率是否顯著低於 95%。此方法直接且適用於比例的假說檢定。
  • Q2, Q3, Q4（兩組存活率差異）：雙比例 z 檢定（Two-Proportion z-test）。比較兩組獨立模擬的存活比例是否有顯著差異。大樣本（10,000 次模擬）確保 z 統計量的常態近似有效。同時報告效果量（存活率差）與 95% 信賴區間。

所有檢定採用顯著水準 alpha = 0.05。由於進行 4 次假說檢定，存在多重比較的偏誤風險，但各假說針對不同面向且彼此獨立，因此不另行 Bonferroni 校正（為避免過度保守）。

To address the four research questions, we employ the following statistical tests:

  • Q1 (Taiwan 4% vs 95%): Binomial Test. Each bootstrap simulation outcome is a Bernoulli trial (survived=1, depleted=0). We test whether the success rate is significantly below 95%. This is the direct and appropriate test for proportion hypotheses.
  • Q2, Q3, Q4 (comparing two survival rates): Two-Proportion z-test. Comparing survival proportions from two independent simulation sets. Large sample sizes (10,000 simulations) ensure the normal approximation is valid. We report effect sizes (survival rate difference) and 95% confidence intervals.

All tests use significance level alpha = 0.05. With 4 hypothesis tests, multiple comparison bias is a concern, but each hypothesis addresses a different dimension and they are independent, so we do not apply Bonferroni correction to avoid over-conservatism.
